# Byte History Site

**Byte History** is a minimal site that shows one historical technology milestone per day.

## Tech

- React 19
- Vite
- Tailwind CSS
- TypeScript

## Requirements

- Node.js >= 18
- npm or pnpm

## Getting Started

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
